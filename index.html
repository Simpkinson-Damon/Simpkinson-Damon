Final Project Submission. This is a Disney Themed app that is based in part on my app
from submission 1. The instructions front page is pretty self explanitory, mouse over the
picture of the theme park you would like information on and click. The name will appear
underneath the picture of the park when you hover over it. You can flip over the cards
after they appear for additional information about that attraction.
It is worth mentioning that this app returns park information in real time so if the park
is closed when you check then you will not get any information from the app.
It is also worth mentioning that sometimes the Themepark API running on the server side
can be slow. If it feels like it is taking too long pick a different theme park and then
come back.
